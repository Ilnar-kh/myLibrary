# myLibrary — консольное приложение для управления библиотекой

Java, Console Application, Collections API

## О проекте

**myLibrary** — простой учебный консольный проект на Java, предназначенный для отработки базовых принципов объектно-ориентированного программирования и работы с коллекциями.

Приложение позволяет пользователю вводить информацию о книгах и хранить её в памяти приложения.

## Описание проекта

Проект реализован в виде консольного приложения.  
Взаимодействие с пользователем происходит через стандартный ввод (`Scanner`).

Приложение демонстрирует базовую структуру backend-приложения:
- сущности (Entity),
- хранилище данных (Repository),
- управляющую логику (Controller).

## Основной функционал

- ввод названия книги;
- ввод имени автора;
- ввод пользовательской оценки книги;
- автоматическая установка даты добавления;
- сохранение книг в памяти приложения;
- вывод списка добавленных книг в консоль.

## Архитектура

Проект построен с использованием простого разделения ответственности:

- **Entity** — модель книги, содержит данные и методы доступа;
- **Repository** — хранение книг в коллекции `ArrayList`;
- **Controller** — точка входа в приложение и логика взаимодействия с пользователем.

Данные хранятся только в памяти и не сохраняются между запусками приложения.

## Хранение данных

- используется коллекция `ArrayList`;
- данные существуют только во время работы приложения;
- база данных не используется.

## Технологический стек

- Java
- Console I/O (Scanner)
- Java Collections Framework

## Назначение проекта

Проект демонстрирует:
- базовые навыки программирования на Java;
- понимание принципов ООП;
- разделение ответственности между классами;
- работу с коллекциями;
- создание простых консольных приложений.
